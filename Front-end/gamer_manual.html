DnD Character Creator – Player Manual

Quick reference for creating, extending, and understanding characters generated by the app.

Home Dashboard Add Game Data

1. Purpose

The app rolls ability scores (4d6 drop lowest), applies Race bonuses (0–2 each), and stores complete characters tied to your account. You can also extend the data set with custom Races, Classes, Abilities, and Weapons.

2. Core Flow

  1. Register / log in.
  2. (Optional) Add new Races, Classes, Weapons, Abilities.
  3. Open character creation page.
  4. Select Race and Class.
  5. Submit: server rolls stats and applies bonuses.
  6. View on dashboard.

3. Ability Scores

Each of the six stats is rolled independently using 4d6 (drop lowest). Racial bonuses are then added. Final stored values already include the bonuses.

Strength

Physical power, melee force.

Dexterity

Agility, reflexes.

Constitution

Endurance, resilience.

Intelligence

Reasoning, knowledge.

Wisdom

Perception, insight.

Charisma

Presence, influence.

Formula: Final = Rolled (top 3 of 4d6) + Race Bonus (0–2)

5. Classes

Contain name, description, and a hit die (e.g. 1d8). Future features might expand this.

6. Weapons & Abilities

Weapons store basic combat fields; Abilities (skills) track a name, description, and associated attribute.

7. Adding Game Data

Use the Add DND Info page. Choose a type and fill the required fields. After submission the entry becomes selectable in character creation.

  • Race Name, description, six bonuses.
  • Class Name, description, hit dice.
  • Ability Name, description, attribute/effect.
  • Weapon Name, type, damage dice, damage type.

8. Example Bonus Application

Rolled Dexterity: 14

Race dexterity_bonus: +2

Final Dexterity Stored: 16

9. Limits

  • No manual rerolls (yet).
  • No negative modifiers.
  • No point-buy / standard array currently.
  • No multiclass support.
  • Equipment not auto-assigned to characters.

10. Troubleshooting

▶Race bonuses all zero?

Recreate the Race ensuring each bonus field is set (0–2).

▶New Race not visible?

Refresh or hard reload to bypass caching.

▶Stats seem low?

Random variance. Future feature may add reroll.

▶Weapon missing?

Ensure all required weapon fields were completed.

11. FAQ

Can I edit a character? Not in current version.

Are bonuses stored separately? Only final totals are persisted.

Can I enter a bonus above 2? Not accepted—stick to 0–2.

Can I view dice rolls? Not yet—only results are shown.

12. Best Practices

  • Pick Race aligning with desired primary stat.
  • Keep custom Race bonuses balanced (0–2).
  • Avoid duplicate naming to prevent confusion.
